## AI Repositories

A collection of notable AI repositories and frameworks.

### Machine Learning Frameworks
- [TensorFlow](https://github.com/tensorflow/tensorflow) - An open-source machine learning framework.
- [PyTorch](https://github.com/pytorch/pytorch) - Tensors and dynamic neural networks in Python with strong GPU acceleration.
- [scikit-learn](https://github.com/scikit-learn/scikit-learn) - Machine learning in Python.
- [Keras](https://github.com/keras-team/keras) - Deep learning for humans.

### Large Language Models
- [llama.cpp](https://github.com/ggerganov/llama.cpp) - Port of Facebook's LLaMA model in C/C++.
- [GPT4All](https://github.com/nomic-ai/gpt4all) - Open-source chatbot ecosystem.
- [Ollama](https://github.com/ollama/ollama) - Get up and running with large language models locally.

### Computer Vision
- [OpenCV](https://github.com/opencv/opencv) - Open Source Computer Vision Library.
- [YOLO](https://github.com/ultralytics/ultralytics) - Object detection and image segmentation.
- [Detectron2](https://github.com/facebookresearch/detectron2) - Facebook AI Research's next generation library.

### Natural Language Processing
- [Transformers](https://github.com/huggingface/transformers) - State-of-the-art Machine Learning for Pytorch, TensorFlow, and JAX.
- [spaCy](https://github.com/explosion/spaCy) - Industrial-strength NLP in Python.
- [NLTK](https://github.com/nltk/nltk) - Natural Language Toolkit.

## Books

Recommended books for learning AI and machine learning.

### Beginner
- **Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow** by Aurélien Géron
- **Python Machine Learning** by Sebastian Raschka
- **Deep Learning for Coders with Fastai and PyTorch** by Jeremy Howard and Sylvain Gugger

### Intermediate
- **Deep Learning** by Ian Goodfellow, Yoshua Bengio, and Aaron Courville
- **Pattern Recognition and Machine Learning** by Christopher Bishop
- **Machine Learning: A Probabilistic Perspective** by Kevin Murphy

### Advanced
- **Reinforcement Learning: An Introduction** by Richard S. Sutton and Andrew G. Barto
- **Probabilistic Graphical Models** by Daphne Koller and Nir Friedman
- **Neural Networks and Deep Learning** by Michael Nielsen (Free online)

### AI Ethics and Society
- **Artificial Intelligence: A Guide for Thinking Humans** by Melanie Mitchell
- **Weapons of Math Destruction** by Cathy O'Neil
- **The Alignment Problem** by Brian Christian

## Contributing

Contributions are welcome! Please feel free to submit a Pull Request. Here are some guidelines:

1. **Add items that are truly awesome** - Only submit resources that you have used and found valuable.
2. **Check for duplicates** - Search existing items before adding new ones.
3. **Follow the format** - Use the existing format for consistency.
4. **Write good descriptions** - Clear, concise descriptions help users understand the resource.
5. **Keep it organized** - Add items to the appropriate category.
6. **Test your links** - Ensure all links work before submitting.
